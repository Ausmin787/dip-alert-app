How to export to PNG

1
Open social-preview.html or readme-banner.html in Chrome at 100% zoom (Ctrl/⌘ + 0).
2
Open DevTools (F12) → ⋮ → More tools → Sensors → set a custom device at the target resolution.
3
Right-click → Inspect the root element → ⋮ → Capture node screenshot.
4
For the Social Preview: upload the PNG in your GitHub repo at Settings → Social Preview.
